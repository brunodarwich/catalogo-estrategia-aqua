# Design System - Catálogo AQUA

Este documento define os padrões visuais e tokens de design para o projeto AQUA, baseados na estética **Quiet Luxury** (Luxo Silencioso).

## 1. Princípios de Design
- **Minimalismo Premium:** Menos é mais. Espaçamento generoso e foco na tipografia.
- **Calma e Confiança:** Uso de cores terrosas e neutras que transmitem exclusividade.
- **Foco no Produto:** A interface serve como moldura para as fragrâncias.

## 2. Design Tokens

### 2.1. Paleta de Cores

| Nome | Hex | Uso Principal |
| :--- | :--- | :--- |
| **Marrom Cafezal** | `#4B3621` | Títulos, Botões Primários, Ícones. |
| **Bege Clássico** | `#F5F5DC` | Fundos de Seção, Bordas. |
| **Creme Suave** | `#FFFDD0` | Fundo Principal da Página. |
| **Ouro Velho** | `#D4AF37` | Detalhes sutis, ícones de destaque. |
| **Branco Off-White** | `#FAF9F6` | Fundos de Modal e Cards. |
| **Texto Secundário** | `#6F4E37` | Descrições e notas olfativas. |

### 2.2. Tipografia

- **Títulos (Serifa):** *Playfair Display* ou *Cinzel*. Transmite tradição e elegância.
- **Corpo e UI (Sans-Serif):** *Montserrat* ou *Inter*. Focada em legibilidade mobile.

| Nível | Família | Tamanho (Base) | Peso |
| :--- | :--- | :--- | :--- |
| H1 (Hero) | Serif | 32px - 48px | 600 |
| H2 (Seção) | Serif | 24px | 500 |
| H3 (Produto) | Serif | 20px | 600 |
| Body | Sans | 16px | 400 |
| Caption | Sans | 14px | 400 |

### 2.3. Espaçamento e Bordas
- **Grid:** Gap de 16px (Mobile) e 24px (Desktop).
- **Padding Global:** 20px nas laterais.
- **Bordas (Radius):** 0px a 4px (evitar bordas arredondadas demais para manter a sobriedade).

## 3. Estados Visuais

- **Active/Hover:** `#3E2C1A` (Marrom mais escuro).
- **Disabled:** `#C0C0C0` (Cinza claro).
- **Focus:** Outline sutil em `#D4AF37`.

## 4. Assets Visuais
- **Ícones:** Linhas finas (Stroke weight: 1px).
- **Imagens:** Filtro levemente quente/saturado para manter a harmonia com os terrosos.
