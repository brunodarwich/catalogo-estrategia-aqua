# design.md
# AQUA — MVP Design System & UX Direction

Versão: 0.1  
Status: MVP Design Direction

---

# 1. Objetivo

Este documento define:

- direção visual,
- UX,
- comportamento da interface,
- componentes,
- estrutura visual,
- motion,
- responsividade,
- e experiência geral do MVP da AQUA.

O objetivo é garantir consistência visual e alinhamento entre:
- branding,
- frontend,
- experiência premium,
- e operação simplificada.

---

# 2. Conceito Visual da Marca

A AQUA deve transmitir:

- sofisticação,
- sensorialidade,
- leveza,
- bem-estar,
- elegância,
- experiência premium,
- modernidade silenciosa.

A interface NÃO deve parecer:
- ERP,
- dashboard empresarial comum,
- ecommerce genérico,
- template pronto,
- sistema técnico pesado.

---

# 3. Direção Estética

A estética deve combinar:

- luxo minimalista,
- design editorial,
- cinematografia,
- interfaces modernas,
- superfícies escuras,
- contraste elegante,
- iluminação suave,
- motion discreto.

---

# 4. Sensação da Interface

A interface deve parecer:

- premium,
- limpa,
- confortável,
- silenciosa,
- sofisticada,
- moderna,
- leve.

# 8. Direção de UX

# Objetivos principais

- poucos cliques,
- baixa fricção,
- simplicidade,
- clareza visual,
- conforto operacional,
- mobile-first.

---

# Regras de UX

## Evitar
- excesso de menus,
- excesso de texto,
- excesso de métricas,
- telas carregadas,
- tabelas complexas.

---

## Priorizar
- espaçamento,
- foco visual,
- leitura rápida,
- ações objetivas,
- hierarquia clara.

---

# 9. Estrutura Mobile-First

Toda a experiência deve ser pensada primeiro para:
- celular.

Depois:
- tablet,
- desktop.

---

# Mobile deve priorizar

- scroll confortável,
- imagens grandes,
- CTAs visíveis,
- leitura rápida,
- navegação simples.

---

# Objetivo

Ser extremamente simples.

- o que está acontecendo,
- o que precisa ser acompanhado,
- e o essencial da operação.

---

- vendas,
- visitas,
- produtos ativos.

---

## Tabelas

# 14. Motion Design

# Direção

As animações devem ser:
- suaves,
- lentas,
- elegantes,
- discretas.

---

# Evitar

- exageros,
- efeitos gamer,
- transições agressivas,
- animações rápidas.

---

# Motion sugerido

- fade,
- opacity,
- blur,
- slide suave,
- parallax leve.
